# Elite102BankingApp
A banking app created with SQL and Python to create a user app that allows for users to create an account, withdraw/deposit recorded in your transactions, view all active accounts, and check your balance in this command line interface. (C2C Project)

# How to Use the Banking App
Launch the program  
Run the Python file. A Tkinter window will appear with buttons for each action, and enter the options in the terminal.

#Choose an option using the Tkinter buttons

- Create Account
- Deposit / Withdraw
- View Active Accounts
- Check Balance
- Wire Transfer

#Follow the prompts in the terminal  
After clicking a button, the program will ask for information in the terminal, such as:

- First name
- Last name
- Account ID
- Deposit/withdraw amount

All confirmations, balances, and account details are printed directly in the terminal.
